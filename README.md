| Input      | Output |
| ----------- | ----------- |
| "green"      | "yellow"       |
| "yellow"   | "red"        |
| "red"      | "green"       |
| "orange"   | nothing        |

This program will cycle the colors from green to yellow, from yellow to red, and from red to green, but only in this order. Any other colors entered into the function will be ignored.
